3）在java中加入布局文件:setContentView(R.layout.first_layout);
4)在注册文件中注册主活动：
5）在java中以id定义按钮：Button 变量名=(Button) findViewById(R.id.他的id);
6)定义按钮活动： 
        protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.first_layout);
        Button button1=(Button) findViewById(R.id.button_1);
        button1.setOnClickListener(new View.OnClickListener(){
            @Override
            public void onClick(View v){
                Toast.makeText(FirstActivity.this,"You clicked Button 1",Toast.LENGTH_SHORT).show();
            }
        });
7)主布局中添加目录： 
    item
              android:id="@+id/add_item"
              android:title="Add"/>
    item
            android:id="@+id/remove_item"
            android:title="Remove"/>
8)在java中创建一定目录： 
     public boolean onCreateOptionsMenu(Menu menu){
        getMenuInflater().inflate(R.menu.main,menu);
        return true;
    }    //这个放在其他函数后面
9）在java中创建目录点击事件： 
       public boolean onOptionsItemSelected(MenuItem item){
        switch (item.getItemId()){
            case R.id.add_item:
                Toast.makeText(FirstActivity.this,"you click Add",Toast.LENGTH_SHORT).show();
                break;
            case R.id.remove_item:
                Toast.makeText(FirstActivity.this,"you click Remove",Toast.LENGTH_SHORT).show();
                break;
            default:
        }
        return true;
    }
